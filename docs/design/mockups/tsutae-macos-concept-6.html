tsutae / macOS concept / warm-soft japanese utility

让语音像快捷键一样安静地工作。

这版不是“AI 工具感”的发光玻璃，而是偏纸感、低饱和、安静克制的 macOS sidecar。重点放在两处：热键触发后的 voice bar， 以及 STT / TTS / fallback 这类工程配置在设置页里的可读性。

02 / System Notification
